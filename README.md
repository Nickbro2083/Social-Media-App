# Social Media App

A mobile social networking application built with **Flutter** and **Firebase**, allowing users to connect, share posts, and interact with content in real time.

# Features

* User registration and authentication
* Create, edit, and delete posts
* View posts from other users
* User profile management
* Real-time data synchronization
* Responsive and user-friendly interface

# Summary

<p align="justify">Social Media App is a mobile application developed using Flutter and Firebase that enables users to create accounts, share posts, and interact with content through a modern and responsive interface. The project was designed to explore mobile application development, cloud-based backend services, and real-time data management.</p>

<p align="justify">Through this project, users can authenticate securely, manage personal profiles, and access social content stored in Firebase. The application demonstrates the integration of Flutter's cross-platform development capabilities with Firebase services such as Authentication, Cloud Firestore, and Storage.</p>

<p align="justify">The project helped strengthen practical skills in mobile development, database design, state management, and Firebase integration while providing hands-on experience in building a complete social networking application.</p>

## Tech Stack

* **Frontend:** Flutter, Dart
* **Backend:** Firebase
* **Database:** Cloud Firestore
* **Authentication:** Firebase Authentication
* **Storage:** Firebase Storage

## Getting Started

### Prerequisites

* Flutter SDK
* Android Studio or VS Code
* Firebase Project

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/social-media-app.git
```

2. Navigate to the project directory:

```bash
cd social-media-app
```

3. Install dependencies:

```bash
flutter pub get
```

4. Configure Firebase for Android/iOS.

5. Run the application:

```bash
flutter run
```
